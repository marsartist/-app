# Weather-app
天气app,功能是获取你所在地的天气情况，以及搜索指定城市的天气，和获取未来几天的天气情况等等，此外该app还添加了记录事情的功能，增删改查等
实现语言：js,css,h5，兼容firefox,chrome以及andriod等，界面如下：
